# Herdr Deck showcase

[Watch or download the 4K60 showcase](./herdr-deck-showcase.mp4)

The video runs for 1 minute 49 seconds and demonstrates project browsing,
workspace launch configuration, project layouts, recent-project toggling,
saved agent sessions, merge-safe worktree cleanup, deletion protection, and
new-directory creation.

The accompanying [shot list](./herdr-deck-showcase-script.md) documents the
performance and timing. Reproducible capture and assembly files are under
[`video-tools`](./video-tools/).
